## Setup

- `:set rnu` 設定相對行號
- `:history`

## Moving

- `<number>H` Go left
- `<number>J` Go down
- `<number>K` Go up 
- `<number>L` Go right
- `<number>G` Go with line number


- `S` Delete line, equals to `cc`


- `"+y` 複製yanked到系統剪貼簿  
- `<CTRL+R>"` 在Command mode貼上yanked  
- `"0p` 貼上前一個複製的內容 



## To Edit

- `i` insert
- `a` append
- `J` Join lines

- `cs"'` 換成對符號


## To Select

- `viB` select block
- `gv` Select back

- `>>` or `<<` 縮排
- `>` or `<` 選取縮排


## To Search


- 單行搜尋字元  `f`/`F` 字元，再按 `;` 或 `,` 重複往後/前找
- 單行搜尋字元到前一個字元  `t`/`T`字元，再按 `;` 或 `,` 重複往後/前找
-  `o`/`O` 往下/上插一行 


## To Delete

- `diw` or `daw` Delete word
